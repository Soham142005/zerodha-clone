# Zerodha Clone - Full Stack Web Project

A functional clone of the Zerodha trading platform built for my BCS 3rd Year project.

## Features
- **Authentication:** JWT-based Login and Signup.
- **Dashboard:** Real-time portfolio tracking (Holdings, Orders, Funds).
- **Responsive UI:** Built with React and Bootstrap.
- **Database:** MongoDB Atlas for storing user stock data.

## Tech Stack
- **Frontend:** React.js, React Router, Bootstrap, CSS Modules.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB.

## How to Run
1. Clone the repo.
2. Run `npm install` in both frontend and backend folders.
3. Start the backend: `npm start`.
4. Start the frontend: `npm run dev`.
